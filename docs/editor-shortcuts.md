# Atalhos do editor

Os atalhos são ignorados enquanto o foco estiver em um campo de texto, seletor
ou input numérico.

| Atalho | Ação |
| --- | --- |
| `B` | Lápis |
| `E` | Borracha |
| `G` | Balde de preenchimento |
| `I` | Conta-gotas |
| `M` | Seleção retangular |
| `W` | Varinha mágica contígua |
| `Q` | Laço livre |
| `D` | Dither |
| `L` | Linha |
| `R` | Retângulo |
| `O` | Elipse |
| `+` / `-` | Aumentar / reduzir zoom |
| `0` | Zoom 100% |
| Roda do mouse | Zoom centrado no cursor |
| `Espaço` + arrastar | Pan do canvas |
| Botão do meio + arrastar | Pan do canvas |
| `Ctrl+Z` | Desfazer |
| `Ctrl+Y` ou `Ctrl+Shift+Z` | Refazer |

Os mesmos atalhos ficam disponíveis dentro do editor em **Ferramentas
profissionais → Atalhos do editor**. O preview do pincel mostra a célula que
será alterada e também os pontos espelhados quando a simetria está ativa.
